Google Search Console 完整教學：免費 SEO 工具這樣用

Travis Chen 2026-05-03 8 分鐘閱讀

如果你只能用一個 SEO 工具，我會毫不猶豫地推薦 Google Search Console（GSC）。不是因為它最強大，而是因為它是 Google 官方的、免費的、而且數據最準確的。

Ahrefs、SEMrush 那些付費工具的數據都是「預估」的，只有 Google Search Console 才能告訴你：你的網站在 Google 搜尋中真正被搜尋了幾次、排在第幾名、被點了幾次。

一、設定與驗證

使用 Google Search Console 的第一步是驗證你擁有這個網站。Google 提供幾種驗證方式：

  1. HTML 檔案上傳 — 把一個驗證檔案放到網站根目錄，最簡單直接
  2. HTML meta 標籤 — 在首頁的 <head> 加入一段驗證 meta 標籤
  3. DNS 記錄 — 在域名的 DNS 設定加入一筆 TXT 記錄，適合沒有網站管理權限但有域名管理權限的情況
  4. Google Analytics 串接 — 如果已經裝了 GA，可以直接用 GA 驗證

在 HEY!BOSS 銀月數位顧問建置的網站，上線時就已經自動完成 GSC 驗證和 Sitemap 提交，客戶不需要自己處理這些技術細節。

二、URL 檢查工具

這是 GSC 最常用的功能之一。在搜尋框輸入任何一個你的網頁 URL，Google 會告訴你：

  • 這個頁面有沒有被 Google 索引？ — 如果沒有，Google 會告訴你原因（例如被 robots.txt 擋住、有 noindex 標籤等）
  • 上次爬取的時間 — 知道 Google 多久來看你的網站一次
  • 行動裝置可用性 — 這個頁面在手機上有沒有問題
  • 結構化資料 — 你的 JSON-LD 結構化資料有沒有被正確讀取

如果你剛發佈了一篇新文章或修改了頁面內容，可以用 URL 檢查工具的「要求建立索引」按鈕，請 Google 優先來爬取這個頁面，不用等它自己慢慢來。

三、效能報告：SEO 的核心數據

效能報告是 GSC 最有價值的功能。它顯示四個關鍵指標：

  • 曝光次數（Impressions） — 你的網頁在搜尋結果中被看到幾次
  • 點擊次數（Clicks） — 被點擊進入的次數
  • 點擊率（CTR） — 點擊次數 / 曝光次數，反映你的標題和描述吸不吸引人
  • 平均排名（Position） — 你在搜尋結果中的平均排名位置

怎麼用這些數據？

最實用的分析方式：

  1. 找到高曝光低點擊的關鍵字 — 這表示你排名不錯但標題不夠吸引人，改善標題和 meta description 就能大幅提升流量
  2. 找到排名 5-15 的關鍵字 — 這些是最有機會衝上首頁的關鍵字，針對性地優化這些頁面的內容
  3. 觀察趨勢 — 哪些關鍵字的排名在上升？哪些在下降？下降的可能需要更新內容

四、Sitemap 提交

Sitemap（網站地圖）是一個 XML 檔案，列出你網站上所有希望 Google 索引的頁面。提交 Sitemap 等於是主動告訴 Google：「這些是我的重要頁面，請來看看。」

提交方式很簡單：

  1. 在 GSC 左側選單找到「Sitemap」
  2. 輸入 Sitemap 的 URL（通常是 https://yourdomain.com/sitemap.xml）
  3. 點擊提交

提交後，GSC 會顯示 Sitemap 中有多少頁面被成功索引、多少有錯誤。如果有錯誤，會具體告訴你是哪些頁面、什麼問題。

五、其他實用功能

涵蓋範圍報告

一次看你所有頁面的索引狀態。如果有大量頁面出現「已排除」，需要檢查是不是網站結構出了問題。

行動裝置可用性

列出所有在手機上有問題的頁面，例如文字太小、可點擊元素太靠近、內容超出螢幕寬度等。

連結報告

顯示外部連結（哪些網站連到你）和內部連結（你自己的頁面之間怎麼互連）。外部連結是 SEO 排名的重要因素之一。
